## 三、关键缺失汇总

### 🔴 必须补充的强缺失项（直接阻塞模型训练）

| 缺失参数             | 符号        | 缺失原因            | 对模型的影响                                         |
| ---------------- |:---------:| --------------- | ---------------------------------------------- |
| **水温**           | `T`       | CSV 中无温度字段      | 所有 μ 和 b 参数都随温度呈指数变化，不可缺                       |
| **剩余污泥排放量**      | `Q_WAS`   | CSV 中无此字段       | SRT 完全依赖此参数，b_H、b_A 无法校准                       |
| **外回流量**         | `Q_RAS`   | CSV 中无此字段       | 影响生物量平衡，MLSS 的物料衡算缺此项会出错                       |
| **出水/末端可溶性 COD** | `S_S,out` | CSV 中无出水 COD 字段 | PINN 的 Loss 函数核心标签之一，缺失则异养参数 μ_H、K_S 无法从数据中被辨识 |
| **反应器有效容积**      | `V`       | 设计常数，不在 SCADA 中 | 常数但必须补充，可从工程设计资料获取                             |

### ⚠️ 可通过换算/近似处理的项（有条件可用）

| 参数              | 现有数据           | 换算/近似方法         |
| --------------- | -------------- | --------------- |
| 内回流实际流量 `Q_IR`  | 内回流频率（一期南/北）   | 需要泵的流量-频率标定曲线换算 |
| 氧传质系数 `KLa`     | 风机风量（1#/2#/3#） | 需要曝气模型换算        |
| 挥发性悬浮固体 `MLVSS` | MLSS（南/北）      | MLSS × 0.75 估算  |
| 池内氨氮/硝氮（近似出水值）  | 一期北/南氨氮、硝氮     | 可近似代替最终出水值使用    |
